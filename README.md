# FootConnect ⚽

A football community platform built using React, Node.js, Express, and PostgreSQL.

## Features

### Clubs

* Create and manage football clubs
* Browse clubs
* Join club requests
* Club notifications

### Friendlies

* Send friendly match requests
* Accept or decline requests
* View upcoming friendlies
* Friendly match notifications

### Home Page

* Upcoming friendlies section
* Community statistics

## Tech Stack

### Frontend

* React
* Axios
* Material UI

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL
