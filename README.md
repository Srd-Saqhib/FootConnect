# FootConnect ⚽

FootConnect is a football community platform that connects players and clubs across different regions. The platform allows clubs and players to discover each other, build connections, organize friendly matches, participate in tournaments, and engage with the football community.

## Live Demo

Frontend: https://foot-connect-eta.vercel.app

Backend API: https://footconnect-backend-yax2.onrender.com

---

* Player Registration & Login

* Persistent Login Sessions

* Browse Football Clubs
* View Detailed Club Profiles

* Club Statistics

* Browse Player Profiles
* Connect With Other Players

* Player Connection Requests

* Decline Friendly Requests
* Friendly Match Notifications

### Tournaments

## Architecture

Frontend (React + Vite)
↓
Backend API (Node.js + Express)
↓
PostgreSQL Database

Socket.IO is used for real-time notifications and updates.

## Installation

### Clone Repository

```bash
git clone <repo-url>
cd FootConnect
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

### Backend

```bash
cd Backend
npm install
npm start
```
